# MX8 Mintlify Docs

## Run locally

```bash
npm i -g mint
cd mintlify-docs
mint dev
```

Then open `http://localhost:3000`.

## Core files

- `docs.json` — Mintlify config and navigation
- `index.mdx` — overview
- `quickstart.mdx` — first job
- `transforms.mdx` — transform workflows
- `api.mdx` — API and job status surface
- `support.mdx` — support matrix
- `onboarding.mdx` — design-partner onboarding
